# Thesis Coordination System

A web-based Thesis Coordination System designed to help students, supervisors, and admins manage thesis topics, submissions, evaluations, and communications in an academic environment.

## 🧠 Project Overview

This system supports a structured workflow for undergraduate/graduate thesis management by providing distinct interfaces for:

- **Students** — select and submit thesis work
- **Supervisors** — approve/assign topics, evaluate submissions
- **Admins** — manage users and system settings

It aims to streamline the thesis process by minimizing manual coordination and reducing paperwork.

## 📂 Features

- 📌 Role-based access (Admin, Supervisor, Student)
- 📌 Topic selection and assignment
- 📌 Submission tracking
- 📌 Supervisor feedback and evaluation
- 📌 User management dashboard
- 📌 Responsive web UI

## 🛠️ Technology Stack

The project includes (but is not limited to):

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Database:**  MongoDB
